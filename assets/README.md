Place student pixel art here.

If a PNG file is named in `game-config.js` with `image: "file-name.png"`, the game uses it automatically.
If it does not exist, the game keeps using the temporary Canvas drawing.

Default file names:

- fisherman.png
- cliffs.png
- mel_peixe.png
- fredy_peixe.png
- peixe_andreia.png
- peixe_azul.png

Tips:

- Use PNG files.
- Transparent background is best.
- Draw fish facing right. The game flips them when they swim left.
- Avoid spaces, accents and uppercase letters in file names.
- For a new fish, upload the PNG here and write its file name in `game-config.js`.
