# Project instructions for Codex

This is a small school browser game called "Maré Pixel: Pesca da Nazaré".

## Main goal

Create a simple HTML/CSS/JavaScript Canvas game for children to play in the browser.

The game must stay simple and classroom-friendly.

## Technology rules

Use only:

- HTML
- CSS
- JavaScript
- Canvas

Do not add:

- React
- Vite
- npm
- Node.js
- Firebase
- backend servers
- databases
- login systems
- online accounts
- student data collection

## Game concept

The player controls a fishing hook with the mouse.

Fish, jellyfish and trash move through the sea.

When an item touches the hook, it attaches to the hook.

While carrying an item, the player cannot catch another item.

The player must pull the hook to the top water line to collect the item.

Fish give points.

Jellyfish remove points.

Trash gives zero points unless a specific mode changes that later.

## Style

Use a pixel-art-friendly style.

Students will later replace placeholder drawings with their own assets in the assets folder.

Keep code readable and beginner-friendly.

## Validation

Because this is a static browser game, there is no build step.

To test, open index.html in a browser or use GitHub Pages after pushing.
